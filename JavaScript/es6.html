# 1. es5拓展

# 1.1 es5 严格模式

  • 目的：
    • 规范化代码，一些不符合规范的代码将不能被执行
  • 使用：
    • 在开头加上"use strict"; ，注意有引号
  • 特点：
    • 未声明的变量不能被使用，会报错
    • 禁止this指向全局对象，原来指向全局对象的改为 undefined
    • 调用构造函数必须加 new
    • 对象如果有重名属性，将报错。（原本是最后写的覆盖前面的）
    • es6引入了块级作用域，为此严格模式只允许在全局作用域或函数作用域的顶层声明函数
    • 新增关键字：let、yield、implements、interface、package、private、protected、public、static

# 1.2 JSON对象

  • json字符串：

    • json数组
    • json对象

  • JavaScript对象和数组与json字符串之间的转换

    • js对象（数组）--> json对象（数组）

      JSON.stringify(obj/arr);

    • json对象（数组）--> js对象（数组

      JSON.parse(json);

# 1.3 Object的拓展

新增方法：

  • 1）Object.create(原型对象，新加属性/方法)
    • 应用：
      • 继承对象的属性和方法
    • 作用：
      • 以指定对象为原型，创建一个新对象
    • 参数：
      • 1）指定对象，将作为新创建对象的原型对象
        • 新对象可以通过 __proto__ 访问到该原型对象
      • 2）新加属性/方法（可选），在新对象中添加新的属性和方法
        • 传入的是一个对象，对象里的属性要传一个关于该属性的描述对象
var classA = {
    class: a,	//枚举实例时也会遍历到该属性
}
var studentA = Object.create(classA,{
    name:{
        value: 'korilo',		//name属性的属性值
        writable: false,		//是否可修改（false为只读），默认为false
        configurable: true,		//是否可删除，默认为false
        enumerable: true		//是否可枚举（被 for in 遍历），默认为false
    }
});
studentA.name = 'chendabin';	//报错，Cannot assign to read only property
  • 2）Object.defineProperty(实例, 属性 , 属性描述)
    • 作用：
      • 对创建的实例进行修改
    • 参数：
      • 1）实例对象，需要修改属性值的实例
      • 2）需要修改的属性/方法
      • 3）修改内容的描述对象
Object.defineProperty(studentA, name, {
        value: 'chendabin',		//name属性的属性值
        writable: false,		//是否可修改
        configurable: true,		//是否可删除
        enumerable: true		//是否可枚举（被 for in 遍历）
});
  • 3）Object.defineProperties(实例, 属性+属性描述)
    • 作用：
      • 对创建的实例进行修改
    • 参数：
      • 1）实例对象，需要修改属性值的实例
      • 2）需要修改的属性/方法以及修改内容的描述对象
        • 修改的属性会在没有修改的属性前面插入
var studentA = Object.defineProperty(studentA, {
    name:{
        value: 'chendabin',		//name属性的属性值
        writable: false,		//是否可修改
        configurable: true,		//是否可删除
        enumerable: true		//是否可枚举（被 for in 遍历）
    }
})

对象字面量增强：

  • 1）get
    • 作用：
      • 读取该属性值时调用函数
      • 返回值作为该属性的属性值
  • 2）set
    • 作用：
      • 当修改该属性时调用函数
      • 注意： 若没有set属性描述，对属性进行赋值不会进行任何操作
    • 参数：
      • data：此次赋值操作赋予的属性值
    • 应用：
      • 反赋值操作：
        • 可以在赋值时输入特定格式的字符串，调用函数对字符串进行修改
        • 将修改的结果赋值给其他属性

参考：

var a = {
    firstName: 'chen',
    lastName: 'dabin',
    fullName:{
        //监听，当获取属性时触发
        get: function(){
            return this.firstName + this.lastName;
        },
        
        //监听，当修改属性时触发
        set: function(data){
            console.log('不要设置我!!!');
            
            //反赋值操作
            if( data.split('-').length === 2 ){
                this.firstName = data.split('-')[0];
            	this.lastName = data.split('-')[1];
            }else{
                console.log('输入格式不规范，请输入"姓-名"格式的名字');
            };
            
        }
    }
};
console.log(a.fullName); 	//chendabin
a.fullName = 'lu-benwei';	// 不要设置我
console.log(a.fullName); 	//lubenwei
  • get 和 set 的另一种写法
    • 实际上面的例子中，设置 get 和 set 的属性是对象里的一个方法
    • 调用的是 get 和 set 的函数，类似于两个事件函数
    • 可以用函数的写法来对fullName方法添加监听事件

参考：

var a = {
    firstName: 'chen',
    lastName: 'dabin',
    get fullName(){
        return this.firstName + this.lastName;
    },
    set fullName(data){
            if( data.split('-').length === 2 ){
                this.firstName = data.split('-')[0];
            	this.lastName = data.split('-')[1];
            }else{
                console.log('输入格式不规范，请输入"姓-名"格式的名字');
            };
    }
}

# 1.4 数组的拓展

新增的方法：

  • 1）数组.indexOf(value)
    • 作用：
      • 返回value在数组里第一次出现的索引
      • 没有找到时返回 -1
  • 2）数组.lastIndexOf(value)
    • 作用：
      • 返回value在数组里最后一个出现的索引
      • 没有找到时返回 -1
  • 3）数组.forEach(function(item, index){})
    • 作用：
      • 对数组元素遍历并调用回调函数
    • 注意：
      • 返回值为undefined
      • 不会修改原数组
    • 回调函数参数：
      • 1）item：数组元素
      • 2）index：元素的索引
  • 4）数组.map(function(item, index){})
    • 作用：
      • 对数组元素遍历并调用回调函数，最后返回加工后的新数组
      • 返回值为回调函数返回值所组成的数组
    • 注意：
      • 没有经过处理的值将返回undefined
      • 不会修改原数组
    • 回调函数可选参数：
      • 1）item：数组元素
      • 2）index：元素的索引
  • 5）数组.filter(function(item, index){})
    • 作用：
      • 对数组元素遍历并调用回调函数，最后返回加工后的新数组
      • 返回值为回调函数返回值为true所对应的部分原数组
    • 注意：
      • 回调函数必须返回一个boolean值
      • 回调函数返回值为false则不返回该元素（有隐式转换，转换为boolean）
      • 不会修改原数组
    • 回调函数可选参数：
      • 1）item：数组元素
      • 2）index：元素的索引
  • 6）数组.reduce(function(preValue, next){}，initValue)
    • 作用：
      • 对数组中的所有内容进行汇总
      • 返回值为回调函数汇总操作的结果
    • 参数：
      • 1）回调函数
      • 2）初始化的值
    • 注意：
      • 回调函数必须返回一个值
      • 回调函数一般为数组元素汇总的操作，如相加，相乘等
      • 不会修改原数组
    • 回调函数可选参数：
      • 1）preValue：
        • 第一轮调用：第一个数组元素
        • 第二轮以后：每次回调返回的值
      • 2）next：
        • 下一个数组元素

# 1.5 函数拓展

新增的方法：

  • 1）call( )方法

    • Function原型对象的方法
    • 语法：
      • 函数.call(对象，函数形参...)
    • 作用：
      • 让一个指定对象强制调用指定函数，并传参
    • 适用函数：
      • 需要传参的函数
    • 应用：
      • this重定向
    var a = {
    fn: function(){
        function show(){
            console.log(this);
        }
        show.call(this);	//让this强制调用show函数,
        					//结果就是改变了show函数里的this
        					//从window变为了外层的a对象
    }
}
  • 2）bind( )方法
    • Function原型对象的方法
    • 语法：函数.bind(对象)
    • 作用：
      • 让一个指定对象强制作为指定函数的调用对象
      • 此时指定函数没执行，只是进行了绑定，相当于只赋值了this
      • 多用于回调函数中，解决this重定向问题
    • 适用函数：
      • 不需要传参的函数
    • 应用：
      • this重定向
var a = {
    fn: function(){
        function show(){
            console.log(this);
        }
        show.bind(this)();	//让this强制作为show函数的调用对象，并执行,
        					//结果就是改变了show函数里的this
        					//从window变为了外层的a对象
    }
}
  • 3）apply( )方法
    • Function原型对象的方法
    • 语法：
      • 函数.apply(对象，数组)
      • 数组里装的是函数的形参
      • 函数.apply(对象，[形参1,形参2...]) === 函数.call(对象，形参1,形参2...)
    • 作用：
      • 让一个指定对象强制调用指定函数，并传参，参数在数组里
    • 适用函数：
      • 一些无法以数组为参数的函数
        • 如：Math.MAX.apply(null, [0,15,26,32,15] )
    • 应用：
      • this重定向
var a = {
    fn: function(){
        function show(){
            console.log(this);
        }
        show.apply(this);	//让this强制调用show函数,没有传参
        					//结果就是改变了show函数里的this
        					//从window变为了外层的a对象
    }
}

# 2. ES6变量声明与赋值

# 2.1 块级作用域

  • 概念：
    • 大括号里的内容

# 2.2 var 变量

  • 特点：
    • 在一般的JavaScript里没有块级作用域的概念
    • var声明的变量只要不是在函数里声明的，都属于全局变量
//区块里用var声明的变量是全局变量

var a = 0;
{
    var a = 1;
}
console.log(a);		//1

# 2.3 let变量

  • 特点：
    • es6加入的新的变量声明方式
    • 用let声明的变量仅在块级作用域（局部）起作用
    • 用let命名不会造成全局命名空间的污染，可以防止数据污染
//区块里用let声明的变量不会对外部的变量起作用

var a = 0;
{
    let a = 1;
}
console.log(a);		//0
  • 注意：
    • 用let声明的变量不能被提前声明
    • 若声明前有其他函数要调用该变量，将报错，不会输出undefined
var a = 0;
{
    console.log(a);		//报错
    let a = 1;
    console.log(a);
}
console.log(a);		//0
  • 应用：
    • 经典循环赋值函数问题:
      • 用for循环对按钮进行事件绑定，当点击时输出该btn的的下标
    • 最简单的解决方法：
      • 用 let 代替 var 声明变量 i
      • 当触发事件响应函数时，闭包里保存的是let声明变量的值
//常见陷阱：
var btn = document.getElementByTagName('button');
for(var i = 0; i < btn.length; i++){
    btn[i].onclick = function(){
        console.log(i);
    }
}
//无论按哪个按钮，输出的都是数组的长度


//解决：
var btn = document.getElementByTagName('button');
for(let i = 0; i < btn.length; i++){
    btn[i].onclick = function(){
        console.log(i);
    }
}

# 2.4 const 常量

  • 特点：
    • es6加入的新的变量声明方式
    • 用lconst声明的变量无法被修改值
    • 和let一样只在块级作用域起作用
    • 无法被再次定义，有点类似严格模式下的变量定义
    • 命名规则：一般规定命名为全大写
    • 声明的是对象时，可以修改对象的属性和方法，引用地址不能被修改

# 2.5 解构赋值

  • 作用：

    • 简略赋值操作

  • 语法：

    • 数组结构赋值：

      • 会拆解数组并分别对同下标的变量赋值
      let [a,b,c] = [1,2,3];

    • 字符串解构：

      • 与数组类似，可快速拆解字符串
      let [a,b,c] = 'abc';

    • 对象解构赋值：

      • 可以快速获取对应属性值
      let people = {
    nage: 'lubenwei',
    age: 18,
    gender: 'male'
}
let {name,age,gender} = people;
console.log(name,age,gender);	//lubenwei 18 male

  • 数组解构和对象解构的区别：

    • 数组和字符串解构是按顺序来匹配的
    • 对象解构是按属性名来匹配的

  • 对象结构属性不匹配的情况

    • 不匹配的为undefined

  • 数组解构参数数量不等的情况：

    • 左边参数多于右边：

      • 没有赋值的位undefined
      • 左边可以设置默认值，没有赋值时为默认值
      • 右边undefined对应的是默认值，null对应的是null
      //默认值
let [a,b,c = 3] = [1];
console.log(a,b,c);		//1, undefined, 3

//undefined和null
let [a,b,c = 3] = [1,null,undefined];
console.log(a,b,c);		//1, null, 3

    • 左边参数少于右边

      • 取右边相等数量的参数参与赋值

  • 注意：

    • 属性在解构之前就被赋值，之后解构的时候会报错
    • 解决方法：将解构语句外包一个括号
    var a = 1;
( [a,b,c] = [1,2,3] );

# 3. 函数拓展

# 3.1 for封装

1）for in循环

数组：

for(let i in people){
    console.log(i);				//index
    console.log(people[i]);		//value
}

对象：

for(let i in people){
    console.log(i);				//key
    console.log(people[i]);		//value
}

2）for of 循环

  • 应用：
    • 遍历数组
    • 通过对数组的解构可以快速获取Map对象的key和value值
  • 注意：
    • 该方法不能遍历对象
for(let i of people){
    console.log(i);				//value
}

# 3.2 模板字符串

  • 语法：
    • 用两个丶包起来，可以任意拼接字符串
    • 用${}来使用变量、执行函数和其他运算操作
var str = ` 这个苹果${20/5}元一斤 `;

# 3.3 箭头函数

  • 作用：

    • 可以简写函数

  • 语法：

    • 省略function关键字，用箭头连接函数体
    • 若函数中只有一句返回操作，可以直接写在箭头后
var fn = function(a,b){
    return a+b;
}
//箭头函数：
var fn = (a,b)=>a+b;
//等效于：
var fn = (a,b)=>{
    return a+b;
}//多操作时要加大括号
  • 重要应用：
    • 改变函数里面this的指向，与函数外部的this保持一致

# 3.4 函数传参默认值

  • 解决方法：

    • 三元运算符（以前的方法）
    function add(a,b){
    a = a?a:0;
    b = b?b:0;
    return a+b;
}
    • es6新增写法
      • 参数之间可以互相设置默认值，相当于作用域
      • 找不到参数时在外层作用域寻找
      • 在父级作用域找不到参数，则默认值是undefined
    function add(a=0,b=0){
    return a+b;
}
//互相设置默认：
function add(a=0,b=a){
    return a+b;
}

//作用域：
var x = 1;
function add(a=x,b=y){
    return a+b;
}// a默认为1，b默认为undefined

# 3.5 扩展运算符

  • 语法：...变量名

  • 作用：

    • 将数组、字符串或对象进行拆分
    • 一般在外面包一个[ ]，将拆分的内容用数组保存

  • 应用：

    • 1）拷贝数组

      • 拷贝的数组是一个新数组，不等于原来的数组
      • 对新数组进行修改不会修改到原数组
      • 两个数组的地址不同
      var a = [1,2,3,4];
var b = [...a];
console.log(b);		//[1,2,3,4]

    • 2）数组传参

      • 将数组的元素作为参数进行传参时，无需一个个写
      var a = [1,2,3,4];
function fn(a,b,c,d,e){
    console.log(a,b,c,d,e);
}
fn(..a);	//1,2,3,4

    • 3）解构赋值

      var a = function(1,2,3,4){
    return [...arguments];
} 
console.log(a);		//[1,2,3,4]

    • 4）伪数组转换真数组

      var btn = document.getElementByTagName('button');
btn[0].parentNode.onclick = function(event){
    console.log([...btn].indexOf(event.target));
}

# 3.6 剩余运算符

  • 语法：...形参名

  • 与拓展运算符的区别：

    • 拓展运算符在函数的赋值中使用
    • 剩余运算符在函数的声明中使用

  • 作用：

    • 将函数传参的剩余实参作为数组元素保存到形参名的变量中
    function fn(a,...rest){
    console.log(a);
    console.log(rest);
}
fn(1,2,3,4);		//1
					//[2,3,4]
