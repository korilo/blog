# 1. 数组操作

# splice

  • 作用：

    • 替换元素
    • 利用特性可以完成数组的增、删、改、插入操作（万能）

  • 语法：

    • Array.splice(参数1，参数2，...参数3)

  • 参数：

    • 1）开始位置（index）
      • 开始位置的数组元素也会参与操作
    • 2）操作元素的数量
      • 关键参数，可以决定方法的作用
    • 3）替换内容
      • 可以是多个参数

  • 删除操作：

    • 没有第三个参数
    • 第二个参数为要删除的元素数量
    • 不传第二个参数时默认删除开始位置及以后的数组元素，与（index，this.length - index）同效

  • 替换操作：

    • 有第三个参数
    • 也可以理解成删除指定内容后插入元素
    • 会在指定位置替换数组元素
    • 第三个及以后的参数都是替换的数组元素

  • 插入操作：

    • 第二个参数为0
    • 第三个及以后的元素为插入的数组元素
    • 注意插入位置是开始位置前（开始位置的元素会参与操作）

  • 增加操作：

    • 第一个参数是this.length，第二个参数是0
    • 增加操作的意义不大，一般用push更方便，只用传增加的数组元素

# 2. 数字操作

# toFixed

  • 作用：
    • 保留小数位
    • 有四舍五入效果
  • 语法：
    • Number.toFix(参数)
  • 参数：
    • 保留的小数位
