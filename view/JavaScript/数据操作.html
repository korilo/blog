# 字符串操作

# split()

  • 参数: 分隔符

  • 作用:

    • 匹配所有分隔符，在分隔符位置对字符串拆分成数组
    • 不传参时拆分所有字符

# toUpperCase()

  • 作用:
    • 将所有字母转换成大写字母

# charAt()

  • 参数: index(Number)

  • 作用:

    • 返回指定下标字符

# substr()

  • 参数:

      1. 开始下标（包含）
      1. 结束下标（不包含）

  • 作用:

    • 按起始下标截取字符串

# match(String/RegExp)

  • 参数:

    • String: 匹配的字符串
    • RegExp: 匹配规则（正则表达式）

  • 作用:

    • 在原字符串中查找匹配项

  • 返回值: Array/null

    • String:
      • 若匹配到，返回该字符串所组成的数组，数组有index和input属性，分别保存查找到的下标和原字符串
      • 若没匹配到，返回null
    • RegExp:
      • 全局查找: 返回所有匹配到的值所组成的数组，没有特殊属性
      • 局部查找: 返回匹配到的第一个值所组成的数组，数组有index和input属性
      • 元字符匹配: 返回匹配到的字符组成的数组，单值有index和input属性，多值没有
      • 没匹配到: 返回null

  • 注意:

    • 匹配到的值为一项时，返回的数组有index和input属性，分别保存查找到的下标和原字符串
    • 匹配到的值为多项时，返回的数组没有特殊属性
    • 没匹配到时返回null

# 数组操作

# push/pop/shift/unshift(item...)

  • 参数:

    • push: 要插入的数组元素
    • pop: 无,传参会无效
    • unshift: 要插入的数组元素
    • shift: 无,传参会无效

  • 作用:

    • push: 在原数组 后 插入新数组元素参数
    • pop: 弹出最后一个数组元素
    • unshift: 在原数组 前 插入新数组元素参数
    • shift: 弹出第一个数组元素

  • 返回值:

    • push: 新数组的length
    • pop: 弹出的数组元素
    • unshift: 新数组的length
    • shift: 弹出的数组元素

  • 注意:

    • 上述方法会修改原数组
    • 插入整个数组将会把该数组作为一个数组元素插入，可以使用ES6解构运算符先进行解构

# join()

  • 参数: 连接符(String)

  • 作用:

    • 将所有数组元素用连接符拼接成字符串
    • 不传参时直接拼接所有数组元素

  • 返回值:

    • 拼接的字符串

# sort/reverse()

  • 参数:

    • sort: 无，传参会报错
    • reverse: 无，传参会无效

  • 作用:

    • sort: 有序化数组
    • reverse: 反转数组

  • 返回值:

    • sort: 修改后的数组
    • reverse: 修改后的数组

  • 注意: 该方法会修改原数组


# splice(index,count,item...)

  • 参数:

    • index: 开始下标（必须）
    • count: 删除个数（必须）
    • item: 插入的新数组元素（可选）
    • 只传一个参数（Number）时，指定数组长度
      • 小于length: 裁剪数组
      • 大于length: 无效

  • 作用:

    • 插入元素: count设置为0
    • 删除元素: 不设置第三个参数
    • 替换元素: 插入与删除数量相同的数组元素
    • 裁剪数组: 只传一个参数（保留的数组长度）

  • 返回值:

    • 删除的数组部分,并组成新数组

  • 注意: 该方法会修改原数组


# 字符串、数组通用操作

# indexOf()

  • 参数:

    • 查找字符(串)/数组元素

  • 作用:

    • 如果数组（字符串）中有匹配元素，返回第一个查找成功的数组下标（字符串下标）
    • 如果没有该元素，返回-1

  • 应用:

    • 判断数组或字符串中是否有查找元素

# slice()

  • 参数：

    • 开始下标（包含）
    • 结束下标

  • 作用：

    • 返回截取到的字符串/数组

# 对象操作

# Object.assign(obj...)

  • 参数：

    • 需要合并的对象

  • 作用：

    • 将所有参数对象合并并返回得到的新对象
