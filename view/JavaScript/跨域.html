# 同源策略

同源策略是浏览器的安全策略，要求网页中的协议名、域名和端口号必须完全一致

违背同源策略就会产生跨域问题

# 解决方案

  • jsonp
    • 需要前端和后端配置
    • 前端发送callback，值为函数名
    • 后端返回一个fn(data),相当于传值data，并用script标签代替调用函数执行
