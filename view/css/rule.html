# @import

@import "路径"

作用：引入另一个css文件


# @charset

@charset "字符集"

作用：告诉浏览器该css文件使用的字符编码集


# @font-face

@font-face{
    font-family: "字体名";
    src: url();
}

作用：自定义字体，当用到该字体时浏览器会强制让用户下载字体

注意：引用完整的字体可能会造成响应时间过长，影响用户体验，所以自定义字体一般使用不多

拓展应用：字体图标

少量的字体图片集不仅体积小，还可以使用字体的样式，可以提升用户体验

字体图标制作网站：https://www.iconfont.cn/
