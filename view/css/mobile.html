# 1物理像素边框

  • 概念

    • 物理像素
    • css像素
    • 像素比dpr: device-pixel-ratio
      • 物理像素 : css像素

  • 像素比：

    • PC端： 1
    • 移动端：
      • iphone6/7/8: 2
      • iphone6/7/8 plus: 3

  • meta标签

    • name="viewPort"
    • content:
      • initial-scale=1: 缩放比例（1）
      • width=device-width
      • user-scalable=no: 禁止用户缩放页面

# 解决方法一：

设置meta标签的initial-scale

原理：通过缩放像素比来实现1物理像素=1css像素

    function initScale(){
        let dpr = window.devicePixelRatio;
        let scale = 1/dpr;
        let metaNode = document.querySelector('meta[name="viewPort"]');
        metaNode.setAttribute('content',`width=device-width,initial-scale=${scale},user-scalable=no`);
    }
    initScale()
  • 副作用：
    • 所有使用px的属性都会进行缩放
    • 可以使用rem = px * dpr 的方法实现：使用rem的属性还是参照原来的css像素

# 解决方法二：

使用伪元素插入边框，配合媒体查询缩放dpr

原理：通过插入一个定位元素来模拟边框

    .box{
        position: relative;
    }
    .box::before{
        content:'';
        position: absolute;
        left: 0;
        bottom: 0;
        width: 100%;
        height: 1px;
        background: black;
    }
    @media screen and (-webkit-min-device-pixel-ratio: 2){
        .box::before{
            transform: scaleY(0.5);
        }
    }
     @media screen and (-webkit-min-device-pixel-ratio: 3){
        .box::before{
            transform: scaleY(0.33333);
        }
    }

这种方法副作用比较小，也容易实现
