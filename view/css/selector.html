# 选择器

  • 简单选择器：

    • 元素选择器：tag
    • id选择器：#
    • class选择器：.
    • 属性选择器：[]
    • 伪类选择器：:
    • 伪元素选择器：::

  • 特殊选择器：

    • 否定选择器：:not()
    • 结构性伪类选择器：nth/of/empty
    • 链接伪类选择器
    • 表单伪类选择器

  • 选择器组合：

    • 交集选择器：并写
    • 并集选择器：,
    • 后代选择器：空格隔开
    • 子元素选择器：>
    • 下一个兄弟元素选择器：+
    • 后面兄弟元素选择器：~

# 选择器权重

可以计算出一个4位数来表示选择器的权重，权重越高设置样式的优先级也越高

每位数字的相加不会逢十进位（实际上是256进1位）

  • 最高：!important
  • 千位：内联样式1，否则为0
  • 百位：所有id选择器数量
  • 十位：所有类选择器、属性选择器、伪类选择器数量
  • 个位：所有元素选择器、伪元素选择器的数量

当权重相等时，按照顺序比较优先级

# 链接伪类选择器

语法 描述
:link 正常链接（没访问过）
:visited 访问过的链接（由于涉及到用户隐私问题，只能设置字体颜色、背景颜色和边框颜色）
:target 代表一个特殊的元素，一般是a标签跳转的目标（网页url查询元素）
:focus 获取焦点（IE6不支持）
::selection p标签内容选中时（火狐不支持）
::-moz-selection p标签内容选中时（火狐版本）

a标签的伪类选择器有书写顺序要求，顺序不同会造成样式的覆盖

  • 爱恨法则：lvha
    • link
    • visited
    • hover
    • active

# 表单伪类选择器

语法 描述
:enabled 可操作的表单
:disabled 不可操作的表单
:checked 被选中的表单
:focus 被获焦的表单

# 结构性伪类选择器

  • 注意：
    • 传递的index是以1开始的
    • 连写时是并集关系，必须前面加空格
    • 要指定类型时在前面加指定类型名，意思是第几个子元素且是该类型元素
    • 一般配合后代选择器使用
    • xxx-of-type是以元素为为中心，不管class和id
li:nth-child(1)		//是第一个子元素且该元素是li

div li:nth-child(1)		//div后代中是第一个子元素且该是li的元素
语法 描述
:nth-child(n) 第n个且是子元素
:nth-child(odd) 奇数位且是子元素
:nth-child(even) 偶数位且是子元素
:first-child 第一个且是子元素
:last-child 最后一个且是子元素
:only-child 是唯一的子元素
:first-of-type 第一个出现的指定元素
:last-of-type 最后一个出现的指定元素
:nth-of-type(n) 第n个出现的指定元素
:nth-of-type(even) 偶数位出现的指定元素
:nth-of-type(odd) 奇数位出现的指定元素
:only-of-type 唯一一个出现的指定元素
:empty 标签里没有内容（包括空格）的元素
