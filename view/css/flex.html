# 1. 基本构造

  • HTML
<div class="container">
    <div class="item"></div>
    <div class="item"></div>
    <div class="item"></div>
</div>
  • CSS
.container{
    /*子元素为块级元素*/
    display: flex;
    
    /*子元素为内联元素*/
    display: inline-flex;
}

# 2. container属性

# 2.1 flex-direction

  • 主轴的布局和排列方向：
    • row :从左往右的x轴
    • row-reverse : 从右往左的x轴
    • column :从上往下的y轴
    • column-reverse :从下往上的y轴

# 2.2 flex-wrap

  • 主轴换行：
    • wrap：换行
    • nowrap：不换行(默认)，压缩宽度
    • wrap-reverse：副轴反方向换行，类似于主轴为column-reverse的分布
  • 换行规律：
    • 行在副轴上分散分布，由富裕空间填充margin-bottom部分
    • 可以添加align-content来使每行紧贴，将其看做一个整体

# 2.3 flex-flow

  • 设置主轴和换行
    • flex-direction和flex-wrap的简写
    • 默认：row nowrap

# 2.4 justify-content

  • 主轴容器分布：
    • flex-start：开始边靠齐
    • flex-end：结束边靠齐
    • center：居中靠齐
    • space-between：分散排布+两端对齐
    • space-around：分散排布+各个容器有margin且不会被重叠
    • space-evenly：分散排布+各个容器有margin但会被重叠

# 2.5 align-items

  • 侧轴容器分布：
    • flex-start：开始边靠齐
    • flex-end：结束边靠齐
    • center ：居中靠齐
    • baseline：基线对齐
    • stretch ：等高布局（前提：子容器没有高度）

# 2.6 align-content

  • 换行主轴分布（只当有换行时才有效）
    • flex-start：开始边靠齐
    • flex-end：结束边靠齐
    • center ：居中靠齐
    • space-between：分散排布+两端对齐
    • space-around：分散排布+各个容器有margin且不会被重叠
    • space-evenly：分散排布+各个容器有margin但会被重叠
    • stretch ：等高布局（前提：子容器没有高度）

# 3. item 属性

# 3.1 order

  • 设定元素的顺序
    • 默认值0
    • 为元素设定order，将按照升序对元素进行排序显示
    • 序号相同时按照原来的顺序排序

# 3.2 flex-basis

  • 元素初始宽度
    • 默认为auto，即item的宽度
    • 赋值为0：相当于每个元素的初始宽度为0，效果就是按弹性因子平分内容区。
    • 只要小于等于最小元素的宽度，效果都是平分布局（不收缩的情况下）

# 3.3 flex-grow

  • 弹性因子：
    • 数值：正整数
    • 表示item分得剩余空间的比例（拉伸比例）
    • items的宽度小于container宽度时，剩余空间大小为正值
    • 当只有一个item设置了弹性因子时，该元素获得所有的剩余空间
    • 一个以上设置了弹性因子时，设置了弹性因子的item按比分配剩余空间

# 3.4 flex-shrink

  • 收缩因子
    • 数值：正整数
    • 表示item分得剩余空间的比例（收缩比例）
    • items的宽度大于container宽度时，剩余空间大小为负值

# 3.5 flex: 1

  • 等分布局
    • flex-basis、flex-grow、flex-shrink的缩写
    • flex：1
      • flex-basis：0
      • flex-grow：1
      • flex-shrink：1

# 3.6 align-self

  • 垂直对齐方式
    • 类似于vertical-align
      • auto：初始值
      • flex-start：顶端对齐
      • flex-end：底端对齐
      • center：居中对齐
      • stretch：等高对齐（没设置高度）
      • baseline：基线对齐
