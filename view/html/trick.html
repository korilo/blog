# 创建元素快捷键

字符 描述
`*` 快速生成重复元素，后面加数字
`$` 元生成重复内容时的占位符，生成序列
`+` 创建兄弟元素
`>` 创建子元素
`{}` 元素内容部分

# pre标签空格样式

pre标签一般用于显示代码字段，外面套code标签使用

里面的内容会保留原本的格式（空格、缩进等不会被折叠）

pre标签有一个默认样式导致空格不会被折叠：

white-space: pre;

只要加上这个样式，任何文本标签都能使用pre标签的效果

# entity实体字符

html提供了实体字符来显示一些特殊符号

例如： &lt :小于号<

可以防止标签元素的渲染错误

  • 常见的实体字符
字符 显示 描述
& lt < 小于号
& gt > 大于号
& nbsp   空格
& copy © 版权符号

更多实体字符可以查询：实体字符

# a标签功能锚点

  • 作用：

    • 页面跳转
    • 锚点跳转
    • 功能链接

  • 锚点跳转

实际实现原理也是页面跳转，会在链接后面加上跳转字符，已达到锚点跳转的效果

  • 功能链接：
    • js操作
    • 邮件发送
    • 拨号

利用a标签来进行js的相关操作，相当于js的点击事件

<!-- 输出hello -->
<a href="javascript: alert('hello')">按钮</a>

<!-- 取消默认行为 -->
<a href="javascript: return">按钮</a>

利用a标签发送邮件(计算机本地需安装邮箱软件)

<a href="mailto: 123456789@123.com">按钮</a>

利用a标签拨号(计算机本地需安装拨号软件,或使用移动端)

<a href="tel: 15612345678">按钮</a>

# figure标签

  • 作用：

    • 子元素会被认为是图片内容的一部分
    • 类似于form标签、table标签，为图片区域添加语义化

  • 子元素标签：

    • figcaption标签 图片标题

# map标签

  • 作用：

    • 为图片设置点击区域，结合img标签使用
    • map标签要设置name属性，方便链接
    • img标签使用usemap属性链接map

  • 子元素标签：

    • area标签：设置点击区域
    • 属性：
      • shape: circle(圆形)/rect(矩形)/poly(多边形)
      • coords: 描述shape的参数
        • 圆形（圆心x,圆心y,半径）
        • 矩形、多边形（点1x,点1y,点2x,点2y....）
      • href: 链接地址

# 布尔属性

有些元素属性要么不写，要么属性值是属性名，这种属性称为布尔属性

这种属性可以进行简写，不写属性值，默认属性值为属性名

例子：

媒体标签的 autoplay/controls/muted/loop等

input标签选框的checked

# 媒体兼容性

  • 媒体标签

旧版本的浏览器可能不支持video和audio标签，可以在标签中写上内容，提示不兼容的问题

当浏览器不兼容时，会直接显示标签内的文字

  • 媒体文件格式

不同的浏览器支持的音视频格式可能不一致

为了获得更好的兼容性，我们可以用source标签来取代src属性

source标签为媒体标签的子标签，src属性指向不同格式的媒体文件

<video>
    <source src="路径1">
    <source src="路径2">
</video>
  • autoplay属性

部分浏览器可能不支持autoplay属性，因为会吓到用户

可以通过设置解决，也可以加上muted
