HTML：超文本标记语言 (Hyper Text Markup Language)

# 单标签闭合

  • 注意：
    • 在 XHTML、XML 以及未来版本的 HTML 中，所有元素都必须被关闭
    • 即使<br> 在所有浏览器中都是有效的，但使用<br /> 其实是更长远的保障。

# HTML文本标签

  • 文本标签分类：
    • 样式类：如 <big>、<ins>
    • 强调类：如 <em>、<strong>
    • 引用类：如 <q>、<cite>
    • 解释类：如 <abbr>、<acronym>
    • 代码类：如 <code>、<var>
    • 格式类：如<pre>、<kbd>

# a标签作用

  • 作用：
    • 添加页面锚点
      • name属性的应用
      • a标签锚点名，链接时可以通过href="#锚点名"链接到该锚点
    • 跳出框架
    • 发送邮件

# HTML <base>元素

  • 作用：
    • 为页面上的所有链接规定默认地址 或 默认打开方式（target）：
<head>
<base href="http://www.w3school.com.cn/images/" />
<base target="_blank" />
</head>

# HTML字符实体

  • 注意：
    • 实体名称对大小写敏感！
    • 完整的可以查询文档
显示结果 描述 实体名称 实体编号
空格 &nbsp; &#160;
< 小于号 &lt; &#60;
> 大于号 &gt; &#62;
& 和号 &amp; &#38;

# URL 字符编码

  • 作用：
    • URL 编码会将字符转换为可通过因特网传输的格式
    • 完整的可以查询文档
字符 URL 编码
€ %80
£ %A3
© %A9
® %AE

# name属性的应用

  • 表单提交
    • 表单项如果要正确地被提交，每个表单项必须设置一个 name 属性。
  • 设置锚点名
    • a标签锚点名，链接时可以通过href="#锚点名"链接到该锚点
