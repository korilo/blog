    # 1. html架构元素

    语义化标签：标签具有一定的语义在里面，比如h5新增的header标签，它的使用和div标签没有区别，header标签也可以放在页面的尾部以及其他区域。但是header有语义在里面，它明确告诉了代码的使用者这个标签代表的意义

    • 语义化的好处：

      • 使用语义化标签能使结构清晰，方便后期维护
      • 能够利用SEO优化，提高搜索优先级

    • SEO搜索引擎优化

      • 关键词：
        • 一般关键字设置在meta标签里，最多可以设置13个关键词
        • <meta name="Keywords" content="关键词....." />
      • 标签 id ：
        • 可以通过id给标签加上语义，从而方便搜索引擎搜索关键字
      • 语义化标签：
        • 像header这样的语义化标签能够提供更好的关键字检索
        • 优先级比 id 高

    • html5新增语义化标签

      • header
        • 可以在移动端上使用，但pc端最好不用
        • 不支持IE
