# Photoshop基础

‘’ PS ‘’ 是前端使用频率仅次于编辑器的图片处理软件，我们可以通过使用 ps 来完成前端页面的切图，修改设计图等操作

# 常用切图快捷键

功能 快捷键
开启/关闭标尺 ctrl + r
移动视窗 space + 鼠标左键
拖动生成参考线 鼠标在参考线上拉动
放大/缩小视图 鼠标滚轮
删除多余选中 alt+鼠标选中
切图（选中区域） c + enter/双击
导出图片 ctrl + alt + shift + s

# 常用其他功能快捷键

功能 快捷键
新建图形文件 ctrl + n
保存当前图像 ctrl + s
自由变换 ctrl + t
放大/缩小视图 ctrl + '+/-'
合并可见图层 ctrl + shift + e
